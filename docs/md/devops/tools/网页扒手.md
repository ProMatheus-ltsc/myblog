## 操作流程
### 打开软件

![输入图片描述](./images/20220217112330.png)

### 点击File，然后点击 New Project Wizred...，弹出如下界面，选择第一项，点击下一步

![输入图片描述](./images/20220217112422.png)

### 然后在输入框输入你要扒的网站的地址，点击下一步

![输入图片描述](./images/20220217112455.png)

### 选择everything，点击下一步，然后点击完成

![输入图片描述](./images/20220217112526.png)

### 新建文件夹 选择本地新建好的文件夹 然后保存网站源文件的路径后，点击保存

![输入图片描述](./images/20220217112555.png)

### 再单击start，即开始扒网站的文件

![输入图片描述](./images/20220217112624.png)

### 完成扒站

![输入图片描述](./images/20220217112709.png)
